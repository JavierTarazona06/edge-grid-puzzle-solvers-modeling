# Soutenance

- 10 minutes de présentation
- 10 minutes de questions/installation.

## Contenu

La présentation étant courte, il n'est pas nécessaire de rappeler les objectifs du projet ou les règles de vos jeux (vous pourrez les mentionner brièvement lors de la présentation du modèle).

L'important est de présenter :

- la façon dont vous générez vos instances ;
- vos choix de modélisation (modèles PLNE considérés) ;
- le fonctionnement du callback ;
- le fonctionnement de votre heuristique ;
- les résultats obtenus ;
- éventuellement les difficultés rencontrées (modélisation, implémentation, ...).
